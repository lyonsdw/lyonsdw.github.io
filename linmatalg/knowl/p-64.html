The function \(f\colon \R^n\to \R^m\) has \(m\) output components \(y_1,y_2,\ldots,y_m\text{.}\) We can read the limit component-wise

\begin{equation*} \lim_{t\to 0} \frac{\left( y_1(\mathbf{x}_0 + t\mathbf{e}_j) -y_1(\mathbf{x}_0), y_2(\mathbf{x}_0 + t\mathbf{e}_j) -y_2(\mathbf{x}_0), \ldots, y_n(\mathbf{x}_0 + t\mathbf{e}_j) -y_n(\mathbf{x}_0)\right)}{t} \end{equation*}

and then we recognize that the \(i\)th component

\begin{equation*} \lim_{t\to 0} \frac{y_i(\mathbf{x}_0 + t\mathbf{e}_j) -y_i(\mathbf{x}_0)}{t} \end{equation*}

is the partial derivative \(\partial y_i/\partial x_j\text{.}\)
