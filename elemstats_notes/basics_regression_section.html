    This section covers basic problem types in Chapters 8‚Äî12 of Freedman, Statistics [1].

    Subsection 2.3.1 Basic regression problems

    Here are some basic problem types about regression.
    1. The scatter diagram for data called \(X,Y\) shows a linear association, with the following summary statistics.
      \begin{align*} \AVE(X)\amp =20.13 \amp \amp \amp \AVE(Y)\amp =15.86\\ \SD(X)\amp =3.65 \amp \amp \amp \AVE(Y)\amp =4.90\\ \amp \amp r\amp=.82 \amp \amp \end{align*}
      1. Estimate the average \(Y\) value for data whose \(X\) value is near \(24.8\text{.}\)
      2. Estimate the average \(X\) value for data whose \(Y\) value is near \(21.0\text{.}\)
      3. Suppose now that both data sets \(X,Y\) are normally distributed and that the scatter diagram is homoscedastic. For data whose \(X\) value is near \(24.8\text{,}\) estimate the percent of \(Y\) data in the range \(15.86\pm 4.90\text{.}\)
    2. Suppose that data called \(U,V\) have a linear association with a correlation of \(r=-.54\text{,}\) and that the scatter diagram is homoscedastic. Suppose also that \(U,V\) are both approximately normally distributed. Estimtate the average percentile rank for \(V\) data whose \(U\) value is close to 80th percentile.
