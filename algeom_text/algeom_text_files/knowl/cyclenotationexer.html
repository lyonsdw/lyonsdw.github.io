Exercise 5. Cyclic permutations.

A permutation \(\pi\) of a set \(X\) is called a full cycle if the sequence

\begin{equation*} x=\pi^0(x),\pi^1(x),\pi^2(x),\pi^3(x)\ldots \end{equation*}

contains all the elements of \(X\) for some \(x\) in \(X\text{.}\) A permutation \(\pi\) of a set \(X\) is called a \(k\)-cycle if there is a \(k\)-element set \(A\subseteq X\) such that \(\pi\) acts as a full cycle on \(A\) and \(\pi\) acts as the identity on the complement of \(A\) (this means that the sequence \(\pi^0(a),\pi^1(a),\pi^2(a),\ldots\) contains all the elements of \(A\) for some \(a\) in \(A\) and that \(\pi(x)=x\) for all \(x\) in \(X\setminus A\)). For example, the element \(\pi=[1,4,2,3]\) is a 3-cycle in \(S_4\) because \(\pi\) acts on the set \(A=\{2,3,4\}\) by

\begin{equation*} 2\to 4\to 3\to 2 \end{equation*}

and \(\pi\) acts on \(A^c=\{1\}\) as the identity. In \(S_n\text{,}\) we write \((a_1a_2\cdots a_k)\) to denote the \(k\)-cycle that acts as

\begin{equation*} a_1\to a_2\to a_3 \to \cdots \to a_k \to a_1 \end{equation*}

on the set \(A=\{a_1,a_2,\ldots,a_k\}\subseteq \{1,2,\ldots,n\}\text{,}\) where \(a_1,a_2,\ldots,a_k\) are distinct integers. For example, we have \([1,4,2,3]=(2,4,3)\) in \(S_4\text{.}\) Note that this cycle notation is not unique. For example, we have \((2,4,3)=(4,3,2)=(3,2,4)\) in \(S_4.\) A 2-cycle is called a transposition. Now do some examples and prove some things: every permutation is a product of disjoint cycles; every permuation is a product of transpositions; the order of a permutation is the l.c.m. of the lengths of its cycles; more?
