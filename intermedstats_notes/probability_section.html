    Section 2 Probability

    Probability models are mathematical frameworks that help us think about chance processes that produce outcomes that cannot be predicted with certainty. In this section, we begin with one of the simplest types of probability models, in which the set of possible outcomes of a chance process is finite, and for which every outcome has an associated number, called its \(probability\text{,}\) that indicates how likely it is for that outcome to occur in a single repetition of the chance process.

    Subsection 2.1 Finite Probability Models

    Definition 2.1.

    A finite probability function is a function \(p\colon \Omega \to [0,1]\text{,}\) where \(\Omega\) is a finite set, and where \(p\) satisfies
    \begin{equation} \sum_{\omega\in\Omega} p(\omega)=1.\tag{2.1} \end{equation}

    Definition 2.2.

    A finite probability measure is a function \(P\colon \mathcal{P}(\Omega)\to [0,1]\) be given by
    \begin{equation} P(E)=\sum_{\omega \in E} p(\omega)\tag{2.2} \end{equation}
    for \(E\subseteq \Omega\text{,}\) where \(p\colon \Omega\to [0,1]\) is a finite probability function.

    Definition 2.3. Finite probability model.

    A finite probability model is a pair \((\Omega,P)\text{,}\) where \(\Omega\) is a finite set, and where \(P\) is a finite probability measure on \(\Omega\text{.}\) The set \(\Omega\) is called the probability space of the model. Elements of \(\Omega\) are called outcomes, and elements of \(\mathcal{P}(\Omega)\) are called events.

    Subsection 2.2 Vocabulary and properties of probability

    Definition 2.6. More probability terminology.

    Let \((\Omega,P)\) be a finite probability model. The complement \(A^c=\Omega\setminus A\) of an event \(A\) is also called the opposite of \(A\text{.}\) Disjoint events \(A,B\) (that is, \(A\cap B=\emptyset)\) are also called mutually exclusive. Given events \(U,V\) with \(P(V)\neq 0\text{,}\) the conditional probability of \(U\) given \(V\), denoted \(P(U|V)\text{,}\) is defined to be
    \begin{equation} P(U|V)=\frac{P(U\cap V)}{P(V)}.\tag{2.3} \end{equation}
    Events \(U,V\) are called independent if \(P(U\cap V)=P(U)P(V)\text{.}\) Otherwise, \(U,V\) are called dependent.

    Subsection 2.3 Random Samples

    Let \((\Omega,P)\) be a finite probability model with probability function \(p\text{,}\) and let \(n\) be a positive integer. Let \(p_{\Omega^n}\colon \Omega^n\to [0,1]\) be defined by
    \begin{equation} p_{\Omega^n}(\omega_1,\omega_2,\ldots,\omega_n)=p(\omega_1)p(\omega_2)\cdots p(\omega_n).\tag{2.4} \end{equation}
    It is easy to check (see CheckpointÂ 2.10 below) that \(p_{\Omega^n}\) is a probability function. Let \(P_{\Omega^n}\) denote the corresponding probability measure. The probability model \((\Omega^n,P_{\Omega^n})\) is called the space of (random) samples of size \(n\) taken from the space \(\Omega\text{.}\) The space \(\Omega^n\) models the outcomes that are obtained by \(n\) repetitions of the process chance process that produces outcomes in \(\Omega\text{.}\)

    Subsection 2.5 Bayesâ€™ Rule

    Bayesâ€™ Rule can be viewed as a practical method for finding conditional probabilities. For events \(A,B\) in a probability space \(\Omega\text{,}\) where both probabilities \(P(A),P(B)\) are nonzero, the definition of conditional probability (2.3) gives us two ways to write \(P(A\cap B)\text{,}\) namely
    \begin{equation*} P(A\cap B)= P(A)P(B|A) = P(B)P(A|B). \end{equation*}
    Dividing the last two terms by \(P(A)\) leads to the following, which is a basic form of Bayesâ€™ Rule.
    \begin{equation} P(B|A) =\frac{P(B)P(A|B)}{P(A)}\tag{2.6} \end{equation}
    A use case for this version of Bayesâ€™ Rule is a problem in which you know the probabilities in the expression on the right, and you wish to find the probability on the left. In effect, this allows you to use \(P(A|B)\) to find \(P(B|A)\text{.}\)
    ðŸ”—
    In a more nuanced form of Bayesâ€™ Rule, the event \(B\) is one of of a collection of events \(B_1,B_2,\ldots, B_r\) that form a partition of \(\Omega\text{,}\) say, \(B=B_k\text{.}\) In this case, we have
    \begin{align} A \amp = \bigcup_{i=1}^r (A\cap B_i)\tag{2.7}\\ P(A) \amp = \sum_{i=1}^r P(A\cap B_i)\tag{2.8}\\ \amp = \sum_{i=1}^r P(B_i)P(A|B_i)\tag{2.9} \end{align}
    Substituting \(B=B_k\text{,}\) and substituting the last expression (2.9) for \(P(A)\) in (2.6), we have the following form of Bayesâ€™ Rule.
    \begin{equation} P(B_k|A) = \frac{P(B_k)P(A|B_k)}{\sum_i P(B_i)P(A|B_i)}\tag{2.10} \end{equation}
    ðŸ”—
    Terminology comment: Equations (2.8) and (2.9) are sometimes called the law of total probability.

    Subsection 2.6 Discrete Probability Models

    The definitions of probability function and its associated probability measure for finite probability models generalize naturally to countably infinite probability spaces.

    Recall that a set \(X\) is countably infinite if there exists a one-to-one and onto function between \(X\) and the set of nonnegative integers.
    Together, finite probability models and their countably infinite generalizations are called discrete probability models.
    ðŸ”—

    Definition 2.12.

    A probability function on a finite or countably infinite set \(\Omega\) is a function \(p\colon \Omega\to [0,1]\) that satisfies \(\sum_{\omega\in\Omega}p(\omega)=1\text{.}\) A discrete probability measure on a finite or countably infinite space \(\Omega\) is a function \(P\colon \mathcal{P}(\Omega)\to [0,1]\) given by \(P(E)=\sum_{e\in E}p(e)\) for \(E\subseteq \Omega\text{,}\) where \(p\) is a probability function on \(\Omega\text{.}\) A discrete probability model is a pair \((\Omega,P)\text{,}\) where \(\Omega\) is a finite or countably infinite set, and where \(P\) is a discrete probability measure on \(\Omega\text{.}\)

    Subsection 2.7 General Probability Models

    countably infinite spaces are not enough, a little motivation, disclaimer about omitting details
    ðŸ”—
    Infinite sequences of coin tosses. The chance process of flipping a fair coin is modeled by the finite probability space \(\Omega=\{H,T\}\text{,}\) with \(p(H)=1/2=p(T)\text{.}\) A sequence of coin tosses looks like a string of letters \(H,H,T,H,T,T,H,\ldots\text{.}\) We already have the model \(\Omega^n\) for sequences of \(n\) coin tosses, but it is natural to also consider infinite sequences. Letâ€™s call this space \(\Omega^\infty\text{.}\) This space contains an uncountable number of elements. A consequence is that the probability of any given infinite sequence must be zero. Thus, there exists no probability function for \(\Omega^\infty\text{.}\) However, it is still possible to define a meaningful probability measure \(P_{\Omega^\infty}\) for some, but not all, subsets of \(\Omega^\infty\text{.}\) For example, we can say that there is a probability of \(1/4\) that the first two entries of a sequence are a heads followed by a tails. Happily, this probability assignment is compatible with the finite probability measure on \(\Omega^2\text{.}\) We will not address the full details of how we assign a probability measure on the space of infinite sequences of coin tosses. But we will nevertheless use this space in the sections that follow. To work with probability on \(\Omega^\infty\text{,}\) it will be sufficient to use the following intuitive fact. Given an event \(E\subseteq \Omega^n\text{,}\) let \(E'\) be the set of infinite sequences \((\omega_1,\omega_2,\ldots)\) such that \((\omega_1,\omega_2,\ldots,\omega_n)\in E\text{.}\) Then we \(P_{\Omega^\infty}(E')=P_{\Omega^n}(E)\text{.}\)

    Summary. In all cases, finite and infinite, a probability model is completely described by a set of outcomes \(\Omega\) and a probability measure \(P\) that assigns numbers to some, but not necessarily all, subsets of \(\Omega\text{.}\) While we will not give the complete definitions here for non-finite spaces, we will continue to consider both finite and infinite models. Fortunately, we can navigate infinite models using intuition from finite models. Best of all, all of the propositions in this section are valid for all probability models, finite and infinite.
