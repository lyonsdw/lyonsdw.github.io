The set on the right is contained in the set on the left by definition. To show containment in the other direction, let \(g^k\) be an element of \(\langle g\rangle\text{.}\) Use the division algorithm to write \(k=qn+r\) with \(0\leq r\leq n-1\text{.}\) Then we have

\begin{equation*} g^k=g^{qn+r}= (g^{n})^qg^r=g^r. \end{equation*}
