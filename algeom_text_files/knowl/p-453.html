The function \(d\) given by (3.3.3) defines a metric on \(\D\text{.}\) That is, \(d\) satisfies

  1. \(d(z_1,z_2)\geq 0\text{,}\) and \(d(z_1,z_2)=0\) if and only if \(z_1=z_2\)
  2. \(\displaystyle d(z_1,z_2)=d(z_2,z_1)\)
  3. \(d(z_1,z_3)\leq d(z_1,z_2)+d(z_2,z_3)\) (the triangle inequality)

for all \(z_1,z_2,z_3\) in \(\D\text{.}\)
