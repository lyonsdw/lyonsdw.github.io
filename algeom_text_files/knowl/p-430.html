Given \(T\in \H\text{,}\) let \(z_0\in\D\) be the point that \(T\) maps to \(0\text{.}\) It must be that \(T\) maps the symmetric point \(\frac{1}{z^\ast}\) to \(\infty\text{,}\) so \(T\) has the form

\begin{equation*} Tz = \alpha \frac{z-z_0}{z-\frac{1}{z_0^\ast}}= \alpha z_0^\ast \frac{z-z_0}{1-z_0^\ast z} \end{equation*}

for some \(\alpha\text{.}\) A straightforward derivation shows that \(|\alpha z_0^\ast|=1\text{,}\) so that we have (3.3.1) below. Another computation establishes an alternative formula (3.3.2) for \(T\in \H\text{.}\) See ExerciseÂ 3.3.6.1.
