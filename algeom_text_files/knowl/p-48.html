  1. Starting with the right hand side of (1.1.24), we have
    \begin{align*} |z-a|^2 -|a|^2 \amp = (z-a)(z-a)^\ast -aa^\ast\\ \amp = zz^\ast -za^\ast - z^\ast a +aa^\ast -aa^\ast\\ \amp = |z|^2 -2\re(za^\ast) \end{align*}
    as desired. Applying this to the equation given at the end of the part a, we set \(a=-i\) and complete the square by adding \(|-i|^2=1\) to both sides. Rearranging, we have
    \begin{equation*} |z-(-i)|^2 = (\sqrt{6})^2. \end{equation*}
    This is a circle centered at \(a=-i\text{,}\) with radius \(r=\sqrt{6}\text{.}\)
  2. We have
    \begin{align*} \im(az+b) \amp = \im(u+iv)(x+iy) + r+is)\\ \amp = \im( (ux-vy+r) + i(vx+uy+s) \end{align*}
    so the equation \(\im(az+b)=0\) is equivalent to \(vx+uy+s=0\text{.}\) This is the general form for the equation for a line in \(\R^2\text{,}\) as long as \((u,v)\) is not the zero vector (if \(0=u=v\text{,}\) then the solutions to \(s=0\) is either the empty set or the whole plane).
