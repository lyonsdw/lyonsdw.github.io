  1. Because \(G\) is finite, the sequence \(g,g^2,g^3,\ldots\) must eventually repeat, by the pigeonhole principle. Suppose \(g^k=g^\ell\text{,}\) for some \(1\leq k\lt \ell\text{.}\) Then we have \(g^{k-\ell}=e\text{,}\) with \(k-\ell\gt 0\text{.}\) Now that we know that there is some positive integer \(p=k-\ell\) such that \(g^p=e\text{,}\) we may choose \(n\) to be the least positive integer such that \(g^n=e\text{.}\) Now let \(h=g^k\) be an arbitrary element of \(G\text{.}\) Use the division algorithm to write \(k=qn+r\) for some \(r\) in the range \(0\leq r\lt n\text{.}\) Then we have
    \begin{equation*} h=g^k=g^{qn+r}=(g^n)^qg^r = eg^r=g^r. \end{equation*}
    This establishes that \(G=\{g,g^2,g^3,\ldots,g^n\}\text{.}\) We conclude that \(n=|G|\text{,}\) and the proof is complete.
