  1. It is clear by inspection that (3.2.11) sends \(z_1,z_2,z_3\) to \(1,0,\infty\text{,}\) respectively.
  2. It is clear by inspection the following transformations make the correct assignments.
    \begin{align*} \text{case } z_1=\infty \amp : \left[z\to \frac{z-z_2}{z-z_3}\right]\\ \text{case } z_2=\infty \amp :\left[z\to \frac{z_1-z_3}{z-z_3}\right]\\ \text{case } z_3=\infty \amp : \left[z\to \frac{z-z_2}{z_1-z_2}\right] \end{align*}
  3. Solving \(z=\frac{az+b}{cz+d}\text{,}\) we have \(cz^2+z(d-a)-b=0\text{.}\) If \(c\neq 0\text{,}\) we have 1 or 2 solutions
    \begin{equation*} z=\frac{a-d\pm\sqrt{(d-a)^2+4bc}}{2c}. \end{equation*}
    If \(c=0\) and \(d\neq a\text{,}\) we have one solution \(z=b/(d-a)\text{.}\) If \(c=0\) and \(d=a\text{,}\) then we have \(b=0\) and \(Tz=z\text{.}\)
  4. It is clear by inspection that \(V^{-1}Uz_i=w_i\) and also that \(Sw_i=w_i\) for \(i=1,2,3\text{.}\)
