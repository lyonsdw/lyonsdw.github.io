Item a.

  1. Use \(|z-a|^2 = (z-a)(z-a)^\ast\text{.}\) Set the latter expression equal to \(r^2\) and solve for \(z^\ast\text{.}\)
  2. The second equality uses part i. Equality (1) uses invariance of the cross ratio, applying the Möbius transformation
    \begin{align*} u\amp \to u-a^\ast\\ \amp \to \frac{r^2}{u-a^\ast}\\ \amp \to \frac{r^2}{u-a^\ast}+a \end{align*}
    to all four entries in the cross ratio immediately preceding the equals sign labeled (1).
  3. Apply \(T^{-1}\) to the first and last expressions in the derivation.
  4. A complex number \(u\) is real if and only if \(u=u^\ast\text{.}\) Apply this to \(u=az+b\text{.}\) Solving for \(z^\ast\) is straightforward.
  5. This is similar to ii above. Equality (1) uses invariance of the cross ratio, applying the Möbius transformation
    \begin{align*} u\amp \to a^\ast u\\ \amp \to a^\ast u + b^\ast -b\\ \amp \to \frac{a^\ast u + b^\ast -b}{a} \end{align*}
    to all four entries in the cross ratio immediately preceding the equals sign labeled (1).
  6. Apply \(T^{-1}\) to the first and last expressions in the derivation.
