Footnote 1.2.1.

The set of unit quaternions \(U(\H)\) is in one-to-one correspondence with the 3-sphere \(S^3=\{(t,x,y,z)\in \R^4\colon t^2+x^2+y^2+z^2=1\}\text{.}\) This is analogous to the set of norm 1 complex numbers that is in one-to-one correspondence with the 1-sphere \(S^2\text{.}\)
