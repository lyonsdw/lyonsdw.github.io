All three equations can be verified by computing the left and right hand sides using rectangular coordindates. A better approach is to write \(r=|r|(\cos\theta + u\sin\theta)\text{,}\) \(s=|s|(\cos \phi + v\sin\phi)\) in polar form. Then we have \(r^\ast = |r|(\cos\theta - u\sin\theta)\) and \(s^\ast = |s|(\cos \phi -v\sin\phi)\text{.}\) Using (1.2.13), we have
\begin{align*} rs \amp= |r||s|(\cos\theta +u\sin\theta)(\cos\phi+v\sin\phi) \\ \amp = |r||s|\left[\cos\theta \cos\phi - u\cdot v \sin\theta\sin\phi\right]\\ \amp + |r||s|\left[(u\times v)\sin\theta\sin\phi + u\sin\theta\cos\phi +v\cos\theta\sin\phi\right]. \end{align*}
In the last expression above, the real part of \(rs\) is on the first line and the imaginary part is on the second line. Thus we have
\begin{align*} (rs)^\ast \amp = |r||s|\left[\cos\theta \cos\phi - u\cdot v \sin\theta\sin\phi\right]\\ \amp - |r||s|\left[(u\times v)\sin\theta\sin\phi + u\sin\theta\cos\phi +v\cos\theta\sin\phi\right]. \end{align*}
It is straightforward to check that you get the same thing by simplifying
\begin{align*} s^\ast r^\ast \amp= |r||s|(\cos\phi-v\sin\phi)(\cos\theta -u\sin\theta). \end{align*}
This establishes (1.2.15). Doing the same computation with \(s=r^\ast=|r|(\cos\theta - u\sin\theta)\) gives (1.2.16). Finally, use the two equations already established to compute
\begin{equation*} |rs|^2 = (rs)(rs)^\ast = rss^\ast r^\ast = |s|^2 rr^\ast = |r|^2|s|^2, \end{equation*}
then take square roots to establish (1.2.17). A slightly more conceptual approach (assuming you are familiar with the matrix algebra operations of transpose and conjugation) is to first verify that \((M(r^\ast)) = (M(r))^\dagger\text{,}\) where \(\dagger\) denotes the conjugate transpose of a matrix. (1.2.15) follows immediately. Next, verify that \(T(u,v)(T(u,v))^\dagger=(|u|^2+|v|^2)\Id\) (here we are using the same notation used in the solution to CheckpointÂ 1.2.2). This last matrix corresponds to the quaternion \(|r|^2\text{,}\) so this establishes (1.2.16).
