Item d.

First, we explain why it is sufficient to check equality at three points. The right side of (3.6.5) is a Möbius transformation because \(s\circ (h\circ R_{r_{a,b}}\circ h)\circ s^{-1}\) is (by definition) an element of the elliptic group \(\S\text{,}\) we know that every element of \(\S\) is a Möbius transformation by the explanation given in the second paragraph of Subsection 3.4.3. By the Fundamental Theorem of Möbius Geometry, we know that \(T_{a,b}\) and \(s\circ (h\circ R_{r_{a,b}}\circ h)\circ s^{-1}\) are equal if and only if they agree at three points.

Next, we check the input value \(p\text{.}\) We have the following.

\begin{align*} p\amp \stackrel{T_{a,b}}{\longrightarrow} p\\ p\amp \stackrel{s^{-1}}{\longrightarrow} \frac{(\im(b),-\re(b),\im(a))}{\sqrt{1-(\re(a))^2}} \\ \amp \stackrel{h}{\longrightarrow} \frac{(\im(a),\re(b),\im(b))}{\sqrt{1-(\re(a))^2}}\\ \amp \stackrel{R_{r_{a,b}}}{\longrightarrow} \frac{(\im(a),\re(b),\im(b))}{\sqrt{1-(\re(a))^2}}\\ \amp \stackrel{h}{\longrightarrow}\frac{(\im(b),-\re(b),\im(a))}{\sqrt{1-(\re(a))^2}}\\ \amp \stackrel{s}{\longrightarrow} p \end{align*}

Next, we check the input value \(0\text{.}\) We have the following.

\begin{align*} 0\amp \stackrel{T_{a,b}}{\longrightarrow} \frac{b}{a^\ast}\\ 0\amp \stackrel{s^{-1}}{\longrightarrow} (0,0,-1)\\ \amp \stackrel{h}{\longrightarrow} (-1,0,0)\\ \amp \stackrel{R_{r_{a,b}}}{\longrightarrow} (|b|^2 - |a|^2,-2\im(ab),2\re(ab)) \;\;(\text{use } r(-i)r^\ast)\\ \amp \stackrel{h}{\longrightarrow} (2\re(ab),2\im(ab),|b|^2-|a|^2)\\ \amp \stackrel{s}{\longrightarrow} \frac{b}{a^\ast} \end{align*}

Finally, we check the input value \(\infty\text{.}\) We have the following.

\begin{align*} \infty\amp \stackrel{T_{a,b}}{\longrightarrow} -\frac{a}{b^\ast}\\ \infty\amp \stackrel{s^{-1}}{\longrightarrow} (0,0,1)\\ \amp \stackrel{h}{\longrightarrow} (1,0,0)\\ \amp \stackrel{R_{r_{a,b}}}{\longrightarrow} (-|b|^2 + |a|^2,2\im(ab),-2\re(ab)) \;\;(\text{use } rir^\ast)\\ \amp \stackrel{h}{\longrightarrow} (-2\re(ab),-2\im(ab),|a|^2-|b|^2)\\ \amp \stackrel{s}{\longrightarrow} -\frac{a}{b^\ast} \end{align*}
