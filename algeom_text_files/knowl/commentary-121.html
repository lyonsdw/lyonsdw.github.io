Instructor's solution for Exercise 3.3.6.5 (proof of Proposition 3.3.20).

Because \(T\in \HU\) must send the real line to itself, so the preimages \(z_1=T^{-1}(1),z_2=T^{-1}(0),z_3=T^{-1}(\infty)\) must be real. Thus we have

\begin{equation*} Tz=(z,z_1,z_2,z_3)=\frac{z-z_2}{z-z_3}\frac{z_1-z_3}{z_2-z_3}=\frac{az+b}{cz+d} \end{equation*}

for some real \(a,b,c,d\text{.}\) We have

\begin{equation*} Ti = \frac{ai+b}{ci+d}= \frac{ac+bd + i(ad-bc)}{c^2+d^2} \end{equation*}

so \(ad-bc\) must be positive because \(T(\U)=\U\text{.}\)
