Proposition 2.3.6. Characterization of subgroups generated by a set of elements.

Given a set \(A\text{,}\) a word of length \(k\) written with the elements of \(A\) is a formal string of the form

\begin{equation*} w=a_1a_2a_3\cdots a_k \end{equation*}

where each \(a_i\in A\) for \(1\leq i\leq k\text{.}\) For example, the strings \(aab\text{,}\) \(bab\text{,}\) and \(bbb\) are words of length 3 written with the elements of \(A=\{a,b\}\text{.}\) Let \(S\) be a subset of a group \(G\text{,}\) and let \(T=\{s^{-1}\colon s\in S\}\text{.}\) We have the following.

\begin{equation} \langle S\rangle = \bigcap_{H\colon S\subseteq H\leq G}H = \{\text{words written with the elements of} S\cup T\}\tag{2.3.1} \end{equation}
Proof.
