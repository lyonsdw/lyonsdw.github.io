This result is "obvious", so the exercise is about how to say it nicely. Here's one approach: Given \(\pi\) in \(S_n\text{,}\) define a digraph \(G\) with vertex set \(\{1,2,\ldots,n\}\) and where \((i,j)\) is an edge if and only if \(\pi(i)=j\text{.}\) Then the disjoint cycle factors of \(\pi\) are the connected components of \(G\text{.}\) Another proof is by induction. The statement is obviously true for \(S_1\text{.}\) Now suppose the statement holds for \(S_k\) for \(k\lt n\text{,}\) and let \(\pi\in S_n\text{.}\) Let \((1,\pi(1),\pi^2(1), \ldots, \pi^k(1)\) be the \(k\)-cycle that starts with 1. Now apply the inductive hypothesis to \(S_{n-k}\) (with suitable relabelings of indices).
