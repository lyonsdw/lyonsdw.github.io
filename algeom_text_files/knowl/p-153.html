  1. Let \(\omega=e^{i2\pi/n}\text{.}\) Show that \(\omega^k\) is in \(C_n\) for all integers \(k\text{.}\)
  2. Show that, if \(z\) is an element of \(C_n\text{,}\) then \(z=\omega^k\) for some integer \(k\text{.}\)
