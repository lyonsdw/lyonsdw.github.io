Proof of PropositionÂ 2.3.9. For reflexivity of the relation \(\sim_H\text{,}\) we have \(x^{-1}x=e\in H\text{,}\) so \(x\sim_H x\text{.}\) For symmetry, suppose \(x\sim_H y\) so that \(x^{-1}y\in H\text{.}\) Since \(H\) is closed under taking inverses, we have \((x^{-1}y)^{-1}=y^{-1}x\in H\text{,}\) so \(y\sim_H x\text{.}\) For transitivity, suppose \(x\sim H y\) and \(y\sim_H z\text{.}\) Then \(x^{-1}y\) and \(y^{-1}z\) are in \(H\text{.}\) Since \(H\) is closed under the group operation, we have \(x^{-1}yy^{-1}z=x^{-1}z\in H\text{,}\) so \(x\sim_H z\text{.}\) Next, we claim that \([x]=xH\text{.}\) If \(x\sim_H y\text{,}\) then \(x^{-1}y=h\) for some \(h\in H\text{,}\) so \(y=xh\in xH\text{.}\) This shows \([x]\subseteq xH\text{.}\) Conversely, if \(y\in xH\text{,}\) then \(y=xh\) for some \(h\in H\text{,}\) so \(x^{-1}y=h\in H\text{.}\) This shoes \(xH\subseteq [x]\text{.}\)
