Using the same letters as in the solution to Checkpoint 3.2.3, the computation

\begin{equation*} \twotwo{e}{f}{g}{h}\twotwo{a}{b}{c}{c}=\twotwo{r}{s}{t}{u} \end{equation*}

verifies that \({\mathcal T}\) is a homomorphism. It is clear that the scalar matrices are contained in \(\ker{\mathcal T}\text{.}\) To see that \(\ker{\mathcal T}\) is contained in the scalar matrices, observe the Möbius transformation \(\left[z\to \frac{az+b}{cz+d}\right]\) is the identity if and only if \(a=d\neq 0\) and \(b=c=0\text{.}\)
