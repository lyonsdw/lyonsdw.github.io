Instructor's solution for Exercise 3.4.7.2 (proof of Proposition 3.4.19).

Let \(T\in \S\) so that \(T=s\circ R\circ s^{-1}\) for some \(R\in \Rot(S^2)\text{.}\) We have
\begin{align*} d_{\extC}(Tp,Tq) \amp = d_{\extC}(sRs^{-1}p,sRs^{-1}q)\\ \amp = d_{S^2}(Rs^{-1}p,Rs^{-1}q) \;\;(\text{by definition } \knowl{./knowl/elliptic_metric_def.html}{\text{(3.4.7)}})\\ \amp = d_{S^2}(s^{-1}p,s^{-1}q) \;\;(d_{S^2}\text{ is invariant under rotation }R)\\ \amp = d_{\extC}(p,q) \;\;(\text{by definition } \knowl{./knowl/elliptic_metric_def.html}{\text{(3.4.7)}}). \end{align*}
