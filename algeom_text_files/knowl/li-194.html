Item b.
Part (a) says that for all \(k\in K\text{,}\) we have \(gkg^{-1}=k'\) for some \(k'\in K\text{.}\) This is the same as \(gk=k'g\text{.}\) That this holds for all \(k\in K\) means that \(gK\subseteq Kg\text{.}\) A similar argument shows that \(Kg\subseteq gK\text{.}\) We conclude that \(gK=Kg\text{.}\)
