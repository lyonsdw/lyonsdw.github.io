Consequence of this exercise: We have the following explicit formulas for the cross ratio.

\begin{align} (z_0,z_1,z_2,z_3) \amp= \frac{z_0-z_2}{z_0-z_3}\frac{z_1-z_3}{z_1-z_2}\tag{3.2.12}\\ (z_0,\infty,z_2,z_3)\amp = \frac{z_0-z_2}{z_0-z_3}\tag{3.2.13}\\ (z_0,z_1,\infty,z_3)\amp = \frac{z_1-z_3}{z_0-z_3}\tag{3.2.14}\\ (z_0,z_1,z_2,\infty)\amp = \frac{z_0-z_2}{z_1-z_2}\tag{3.2.15} \end{align}
