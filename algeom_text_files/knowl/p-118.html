Given a function \(f\colon X\to Y\text{,}\) there is a natural equivalence relation \(\sim_f\) on \(X\) given by
\begin{equation*} x\sim_f y \Leftrightarrow f(x)=f(y). \end{equation*}
The corresponding set of equivalence classes is \(X/\!\!\sim_f \; = \{f^{-1}(y)\colon y\in f(X)\}\text{.}\) Furthermore, the function \(X/\!\!\sim_f \; \to f(X)\) given by \([x]\to f(x)\) is a one-to-one correspondence.
