Item c.

Let \(d\) be the smallest positive element in \(H=\langle n_1,\ldots,n_r\rangle\text{.}\) By the previous part, \(H=\langle d\rangle\text{,}\) so each \(n_i\) is a multiple of \(d\text{,}\) i.e., \(d\) is a common divisor of the \(n_i\text{.}\) Because every element of \(H\) is a \(\Z\)-linear combination of the \(n_i\text{,}\) there must exist integers \(c_i\) such that \(d=\sum_ic_in_i\text{.}\) If \(d'\) is also a common divisor of the \(n_i\text{,}\) then there exist integers \(k_i\) such that \(n_i=k_id'\text{.}\) Then we have \(d=\sum_i c_i(k_id')=d'\sum_ic_ik_i\text{.}\) Because \(d,d'\) are both positive integers, it must be that \(\sum_ic_ik_i\) is a positive integer, so we conclude that \(d'\leq d\text{.}\) Thus we have proved that \(d=\gcd(n_1,\ldots,n_r)\text{.}\)
