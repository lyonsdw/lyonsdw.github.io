Item b.

First, note that \(\phi(h)=C_h\) does indeed map \(K\) to itself because \(K\) is normal, that \(C_h\) is indeed an automorphism because \((C_h)^{-1}=C_{h^{-1}}\text{,}\) and that \(\phi\) is indeed a homomorphism because \(C_{hh'}=C_hC_{h'}\text{.}\) To see that \(\psi\) is a homomorphism, we have the following.
\begin{align*} \psi((k,h),(k',h')) \amp = \psi (khk'h^{-1},hh')\\ \amp = khk'h^{-1}hh'\\ \amp = khk'h'\\ \amp = \psi((k,h))\psi((k',h')) \end{align*}
The assumption that \(G=KH\) implies that \(\psi\) is onto. If \(e=\psi((k,h))=kh\text{,}\) then \(k=h^{-1}\text{,}\) so \(k\in H\) and \(h\in K\text{.}\) The assumption \(K\cap H=\{e\}\) implies \(k=h=e\text{.}\) We conclude that \(\psi\) is one-to-one.
