Item b.

That \(\phi\) is a homomorphism is clear: we have \(\phi(a+b) = \omega^{a+b}=\omega^a\omega^b=\phi(a)\phi(b)\text{.}\) We have \(\omega^{kn}=\omega^0=1\) for all \(k\in \Z\text{,}\) so \(n\Z\subseteq \ker(\phi)\) Conversely, let \(k\in \Z\) and write \(k=nq+r\) as in the previous part, so that \(\omega^k=\omega^r\text{.}\) Since \(\omega^r=1\) if and only if \(r=0\) for \(0\leq r\leq n-1\text{,}\) we have \(\ker(\phi)\subseteq n\Z\text{.}\) We conclude that \(\ker(\phi)=n\Z\text{,}\) as desired.
