  1. First, check that, for any pure, unit quaternion \(u\text{,}\) we have \(u^2=-1\) (use (1.2.13)). Now we have

    \begin{align*} R_r(u)\amp =(\cos\theta + u\sin\theta)u(\cos\theta + u\sin\theta)^\ast\\ \amp = (\cos\theta + u\sin\theta)(u\cos\theta + \sin\theta)\\ \amp = u(\cos^2\theta+\sin^2\theta) - \sin\theta\cos\theta + \sin\theta\cos\theta\\ \amp = u \end{align*}

  2. We have

    \begin{align*} R_r(v)\amp =(\cos\theta + u\sin\theta)v(\cos\theta + u\sin\theta)^\ast\\ \amp = (\cos\theta + u\sin\theta)(v\cos\theta -vu\sin\theta)\\ \amp = v\cos^2\theta + (uv-vu)\sin\theta\cos\theta -uvu\sin^2\theta\\ \amp =v(\cos^2\theta -\sin^2\theta) + 2w\sin\theta\cos\theta \end{align*}
    (use (1.2.13) to simplify \(uv-vu=2u\times v=2w\) and use ExerciseÂ 1.2.6.1 to get \(v=uvu\text{,}\) the reflection of \(v\) in the plane perpendicular to \(u\)). Using double angle formulas, the last expression in the displayed derivation becomes
    \begin{equation*} v\cos(2\theta) + w\sin(2\theta). \end{equation*}
    Since \(R_r\) is a linear map that acts on the an orthogonal basis \(u,v,w\) as a rotation about \(u\) by the angle \(2\theta\text{,}\) the Proposition is established.
