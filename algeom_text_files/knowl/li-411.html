For (3.6.6), for \(0\lt u\lt 1\text{,}\) we have
\begin{align*} \cosh \left(\ln\left(\frac{1+u}{1-u}\right)\right) \amp = \frac{e^{\ln(\text{stuff})} + e^{-\ln(\text{stuff})}}{2} \;\; \left(\text{stuff }=\frac{1+u}{1-u}\right)\\ \amp = \frac{\frac{1+u}{1-u} + \frac{1-u}{1+u}}{2}\\ \amp = \frac{(1+u)^2 + (1-u)^2}{2(1-a^2)}\\ \amp = \frac{2(1+u^2)}{2(1-a^2)}\\ \amp = \frac{1+u^2}{1-u^2}. \end{align*}
For (3.6.7), using the double angle identity
\begin{equation*} \cos (2\theta) = \cos^2\theta -\sin^2 \theta \end{equation*}
and the identities \(\cos(\arctan u) = \frac{1}{\sqrt{1+u^2}}\text{,}\) \(\sin(\arctan u) = \frac{u}{\sqrt{1-u^2}}\text{,}\) we have
\begin{align*} \cos(2\arctan u) \amp = \cos^2(\arctan u) -\sin^2(\arctan u)\\ \amp = \frac{1}{1+u^2} - \frac{u^2}{1+u^2}\\ \amp = \frac{1-u^2}{1+u^2}. \end{align*}
