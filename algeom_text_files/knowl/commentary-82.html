Instructor's solution for Exercise 2.6.2.

First, reduce \(x\) mod \(p\text{,}\) that is, write \(x=qp+r\) with \(0\leq r\leq p-1\text{.}\) Now consider two cases. The case \(r=0\) is trivial. If \(r\neq 0\text{,}\) apply the fact \(r^{|G|}=e\) (see Exercise 2.3.2.8) to the group \(G=U_p\text{.}\) Since \(p\) is prime, we have \(|U_p|=p-1\text{,}\) so \(r^{|G|}=e\) becomes \(r^{p-1}=1 \pmod{p}\text{.}\) Multiplying both sides by \(r\text{,}\) we have \(r^{p}=r \pmod{p}\text{,}\) as desired.
