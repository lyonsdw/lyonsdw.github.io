Suppose \(H\) is a normal subgroup of \(G\text{,}\) and let \(x,y\) be elements of \(G\text{.}\) To show that \((xH)(yH)\subseteq xyH\text{,}\) let \(u=xh_1yh_2\) for some \(h_1,h_2\) for some \(h_1,h_2\) in \(H\text{.}\) Let \(h_3=y^{-1}h_1y\) (we know \(h_3\in H\) because \(H\) is normal) so that we have \(h_1y=yh_3\text{.}\) Then we have \(u=xyh_3h_2\in xyH\text{.}\) To see that \(xyH\subseteq (xH)(yH)\text{,}\) let \(xyh\in xyH\text{.}\) Because \(H\) is normal, we have \(yh=h'y\) for some \(h'\in H\text{,}\) so we have \(xyh=xh'y=(xh')(ye)\in (xH)(yH)\text{.}\) Conversely, suppose that \((xH)(yH)=xyH\) for all \(x,y\) in \(G\text{.}\) Let \(g\) be an element of \(G\) and let \(h\) be an element of \(H\text{.}\) We have \(ghg^{-1}=(gh)(g^{-1}e)\in (gH)(g^{-1}H)=(gg^{-1})H=H\text{,}\) so \(H\) is normal.
