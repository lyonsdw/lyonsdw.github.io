We have
\begin{align*} \text{circumference } \amp = 2\int_0^{2\pi} \frac{\alpha}{1+\alpha^2}dt\\ \amp = \frac{4\pi\alpha}{1+\alpha^2}. \end{align*}
In terms of \(r=d_{\extC}(0,\alpha)= 2\arctan \alpha\text{,}\) we have \(\alpha = \tan \frac{r}{2}\text{,}\) so
\begin{equation*} \text{circumference } = \frac{4\pi\alpha}{1+\alpha^2} = 4\pi\frac{\tan \frac{r}{2}}{\sec^2 \frac{r}{2}}= 4\pi\sin\frac{r}{2} \cos \frac{r}{2} = 2\pi \sin r. \end{equation*}
