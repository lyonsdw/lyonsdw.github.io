It is easy to see that \(R_r\) is a linear map from the vector space of unit quaternions to itself. That means that the following properties hold for all pure quaternions \(v,w\) and all real scalars \(k\text{.}\)

\begin{gather} R_r(v+w) = R_r(v) + R_r(w)\tag{1.2.17}\\ R_r(kv) = kR_r(v)\tag{1.2.18} \end{gather}
