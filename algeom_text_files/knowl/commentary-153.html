Instructor's solution for Exercise 3.6.6.
  1. Apply Proposition 1.2.9 to \(r_{a,b}=\cos\theta +u\sin\theta\) with \(u=\frac{(\im(a),\re(b),\im(b))}{\sqrt{1-(\re(a))^2}}\) and \(\theta=2\arccos(\re(a))\text{.}\)
  2. Solutions to \(T_{a,b}z=z\) are roots of the quadratic equation
    \begin{equation*} 0 = b^\ast z^2 + z(a-a^\ast) + b. \end{equation*}
    Using \(2i\im(a)=a-a^\ast\text{,}\) we have
    \begin{align*} z \amp = \frac{-2i\im(a) \pm \sqrt{(2i\im(a))^2 - 4|b|^2}}{2b^\ast}\\ \amp = \frac{-2i(\im(a) \pm \sqrt{(\im(a))^2+|b|^2})}{2b^\ast}\\ \amp = \frac{-ib(\im(a)\pm \sqrt{1-(\re(a))^2})}{|b|^2}. \end{align*}
  3. We have
    \begin{align*} s\left(\frac{(\im(b),-\re(b),\im(a))}{\sqrt{1-(\re(a))^2}}\right)\amp = \frac{\im(b) -i\re(b)}{\sqrt{1-(\re(a))^2} -\im(a)}\\ \amp = \frac{-ib}{\sqrt{1-(\re(a))^2} -\im(a)}\frac{\sqrt{1-(\re(a))^2} +\im(a)}{\sqrt{1-(\re(a))^2} +\im(a)}\\ \amp = \frac{-ib(\sqrt{1-(\re(a))^2} + \im(a))}{|b|^2}\\ \amp = p. \end{align*}

  4. First, we explain why it is sufficient to check equality at three points. The right side of (3.6.5) is a Möbius transformation because \(s\circ (h\circ R_{r_{a,b}}\circ h)\circ s^{-1}\) is (by definition) an element of the elliptic group \(\S\text{,}\) we know that every element of \(\S\) is a Möbius transformation by the explanation given in the second paragraph of Subsection 3.4.3. By the Fundamental Theorem of Möbius Geometry, we know that \(T_{a,b}\) and \(s\circ (h\circ R_{r_{a,b}}\circ h)\circ s^{-1}\) are equal if and only if they agree at three points.

    Next, we check the input value \(p\text{.}\) We have the following.

    \begin{align*} p\amp \stackrel{T_{a,b}}{\longrightarrow} p\\ p\amp \stackrel{s^{-1}}{\longrightarrow} \frac{(\im(b),-\re(b),\im(a))}{\sqrt{1-(\re(a))^2}} \\ \amp \stackrel{h}{\longrightarrow} \frac{(\im(a),\re(b),\im(b))}{\sqrt{1-(\re(a))^2}}\\ \amp \stackrel{R_{r_{a,b}}}{\longrightarrow} \frac{(\im(a),\re(b),\im(b))}{\sqrt{1-(\re(a))^2}}\\ \amp \stackrel{h}{\longrightarrow}\frac{(\im(b),-\re(b),\im(a))}{\sqrt{1-(\re(a))^2}}\\ \amp \stackrel{s}{\longrightarrow} p \end{align*}

    Next, we check the input value \(0\text{.}\) We have the following.

    \begin{align*} 0\amp \stackrel{T_{a,b}}{\longrightarrow} \frac{b}{a^\ast}\\ 0\amp \stackrel{s^{-1}}{\longrightarrow} (0,0,-1)\\ \amp \stackrel{h}{\longrightarrow} (-1,0,0)\\ \amp \stackrel{R_{r_{a,b}}}{\longrightarrow} (|b|^2 - |a|^2,-2\im(ab),2\re(ab)) \;\;(\text{use } r(-i)r^\ast)\\ \amp \stackrel{h}{\longrightarrow} (2\re(ab),2\im(ab),|b|^2-|a|^2)\\ \amp \stackrel{s}{\longrightarrow} \frac{b}{a^\ast} \end{align*}

    Finally, we check the input value \(\infty\text{.}\) We have the following.

    \begin{align*} \infty\amp \stackrel{T_{a,b}}{\longrightarrow} -\frac{a}{b^\ast}\\ \infty\amp \stackrel{s^{-1}}{\longrightarrow} (0,0,1)\\ \amp \stackrel{h}{\longrightarrow} (1,0,0)\\ \amp \stackrel{R_{r_{a,b}}}{\longrightarrow} (-|b|^2 + |a|^2,2\im(ab),-2\re(ab)) \;\;(\text{use } rir^\ast)\\ \amp \stackrel{h}{\longrightarrow} (-2\re(ab),-2\im(ab),|a|^2-|b|^2)\\ \amp \stackrel{s}{\longrightarrow} -\frac{a}{b^\ast} \end{align*}
  5. First, verify the suggestions in the hint. Now start in the upper left of the diagram with \(M_{a,b}\text{.}\) Tracing the path to the right and down to the lower right corner, we have
