  1. The points \(A,A'\) are the intersections of great circles \(AC,AB\text{,}\) so they are endpoints of a diameter. (Two planes through the origin intersect in a line through the origin, and that line contains a diameter of the sphere.)
  2. On \(S^2\text{,}\) points \(p,q\) are antipodal if and only if \(q=-p\text{.}\) So we have \(X'=-X\) for \(X=A,B,C\text{.}\) The antipodal map \(x\to -x\) is not an element of \(\Rot(S^2)\text{,}\) but it is clear that it is distance and area preserving. Triangles \(\triangle ABC\) and \(\triangle A'B'C'\) are antipodal images of each other, so they have the same area.
  3. View \(R\) as three 2-gons \(AA',BB',CC'\) that overlap in triangle \(\triangle ABC\text{.}\)
  4. View \(R'\) as three 2-gons \(AA',BB',CC'\) that overlap in triangle \(\triangle A'B'C'\text{.}\)
  5. We have
    \begin{align*} 4\pi \amp =\text{ Area}(S^2)\\ \amp = \text{ Area}(R)+\text{ Area}(R')\\ \amp = 4(\angle A+\angle B+\angle C) - 2\left(\text{Area}(\triangle ABC)+\text{Area}(\triangle A'B'C')\right).\\ \amp = 4(\angle A+\angle B+\angle C) - 4\left(\text{Area}(\triangle ABC)\right) \end{align*}
    Rearranging, we have
    \begin{equation*} \text{Area}(\triangle ABC)= (\angle A +\angle B + \angle C)-\pi. \end{equation*}
