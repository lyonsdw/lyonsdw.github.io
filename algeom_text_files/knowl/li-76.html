Item 3.

From the previous part, we have that every element in \(C_n\) is of the form \(\omega^k\) for some integer \(k\text{.}\) Use the division algorithm to write \(k=qn+r\text{.}\) Then \(\omega^k=\omega^r\text{.}\) We conclude that every element in \(C_n\) is a member of the set
\begin{equation*} \{\omega^r\colon 0\leq r\leq n-1\}. \end{equation*}
