  1. We have \((\omega^k)^n= e^{i2\pi k}=1\text{,}\) so \(\omega^k\in C_n\text{.}\)
  2. Let \(z=\in C_n\text{.}\) Then \(1=z^n = |z^n|= |z|^n\text{,}\) so \(|z|=1\text{.}\) Thus \(z=e^{it}\) for some real \(t\text{.}\) The fact that \(1=z^n=e^{itn}\) means that \(tn=2\pi k\) for some integer \(k\text{.}\) Thus we have \(z= e^{i2\pi k/n}=\omega^k\text{.}\)
  3. From the previous part, we have that every element in \(C_n\) is of the form \(\omega^k\) for some integer \(k\text{.}\) Use the division algorithm to write \(k=qn+r\text{.}\) Then \(\omega^k=\omega^r\text{.}\) We conclude that every element in \(C_n\) is a member of the set
    \begin{equation*} \{\omega^r\colon 0\leq r\leq n-1\}. \end{equation*}
