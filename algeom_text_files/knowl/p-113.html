    1. (reflexivity) It is clear that \(n|(x-x)\) for all \(x\in \Z\text{.}\)
    2. (symmetry) If \(n|(x-y)\text{,}\) say \(x-y=kn\text{,}\) then \(y-x = -kn\text{,}\) so \(n|(y-x)\text{.}\)
    3. (transitivity) If \(n|(x-y)\text{,}\) say \(x-y=kn\text{,}\) and \(n|(y-z)\text{,}\) say \(y-z=\ell n\text{,}\) then
      \begin{equation*} x-z=(x-y)+(y-z)= (k+\ell)n\text{,} \end{equation*}
      so \(n|(x-z)\text{.}\)

The division algorithm says that for any \(x\in \Z\text{,}\) there are unique integers \(q,r\text{,}\) with \(r\) in the range \(0\leq r\leq n-1\text{,}\) such that \(x=qn+r\text{.}\) Thus \(x\sim_n r\text{.}\) This means that \([x]=[r]\) for some \(r\text{,}\) \(0\leq r\leq n-1\text{.}\) The uniqueness of \(r\) means that the equivalence classes \([0],[1],\ldots,[n-1]\) are distinct. We conclude that
    \begin{equation*} \Z_n=\{[0],[1],\ldots,[n-1]\}. \end{equation*}
