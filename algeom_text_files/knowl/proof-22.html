Proof

Proof sketch: The statement about the case when \(C\) contains the north pole is geometrically clear. For the case when \(C\) does not contain \((0,0,1)\text{,}\) choose a rotation \(R\) of \(S^2\) that takes some point on \(C\) to the north pole. Again, let \(T=s\circ R\circ s^{-1}\) be the conjugate element in \(\S\text{.}\) It is clear that \(R\) takes \(C\) to a circle, that \(s\) takes \(R(C)\) to a Euclidean straight line, and that \(T^{-1}\) takes \(s(R(C))\) to a cline (because \(T^{-1}\) is a MÃ¶bius transformation!). Thus \(s(C)=(T^{-1}\circ s\circ R)(C)\) is a cline. Because \((0,0,1)\) is not on \(C\text{,}\) it must be that \(\infty\) is not on \(s(C)\text{,}\) so \(s(C)\) is a circle in \(\C\text{.}\)
