We will use the 2-step subgroup test. First, \(\Inn(G)\) contains the identity map \(C_e\colon G\to G\text{,}\) so \(\Inn(G)\) is not empty. Given \(g,h\) in \(G\text{,}\) we have

\begin{equation*} (C_g\circ C_h)(x) = C_g(C_h(x))=g(hxh^{-1})g^{-1}= (gh)x(gh)^{-1}=C_{gh}(x) \end{equation*}

so \(\Inn(G)\) is closed under composition. We showed in ExerciseÂ 2.4.2.10 that \((C_g)^{-1}=C_{g^{-1}}\text{,}\) so \(\Inn(G)\) is closed under inversion. This completes the 2-step subgroup test, and we conclude that \(\Inn(G)\) is a subgroup of \(\Aut(G)\text{.}\)
