Let \(Tz=\frac{az+b}{cz+d}\text{.}\) Suppose that \(T\) fixes \(0,\infty\text{.}\) We have \(0=T(0)=b/d\text{,}\) so \(b=0\text{.}\) We have \(T\infty = a/c\text{,}\) so \(c=0\text{.}\) Setting \(\alpha = a/d\text{,}\) it must be that \(\alpha\neq 0\) (otherwise \(T\) would be constant) and \(\alpha\neq \infty\) (otherwise \(d=0\text{,}\) which violates the requirement \(ad-bc=0\)). Thus we have \(Tz=az/d = \alpha z\) for some \(\alpha\neq 0\text{.}\) Now suppose \(T\) has a single fixed point at \(\infty\text{.}\) Then \(T\infty = a/c\text{,}\) so \(c=0\text{.}\) Let \(\alpha=a/d,\beta=b/d\text{,}\) so that \(Tz=\alpha z +\beta\text{.}\) If \(\alpha\neq 1\text{,}\) then \(-\beta/\alpha\) is another fixed point, so it must be that \(\alpha=1\text{.}\) If \(\beta=0\text{,}\) then \(T\) is the identity, but that contradicts our assumption that \(T\) has a single fixed point. Thus \(Tz=z+\beta\) for some \(\beta\neq 0\text{.}\)
