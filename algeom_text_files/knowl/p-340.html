For symmetry, we have \(F=\Id F\) for all figures \(F\text{.}\) For reflexivity, if \(F'=gF\text{,}\) then \(F=g^{-1}F'\text{.}\) For transitivity, if \(F'=gF\) and \(F''=hF'\text{,}\) then \(F''=hF'=h(gF)=(gh)F\text{.}\) Comment: This is almost the same exercise as showing that the "in the same orbit" relation coming from a group acting on a set is an equivalence relation. You could even point out that it is the same exercise, if you note that you need to consider the action \(G\to \Perm({\mathcal P}(X))\) (that is, transformations acting on figures) that arises from the action \(G\to \Perm(X)\) of transformations acting on points.
