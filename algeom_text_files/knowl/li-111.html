Item e.
We claim that \(\Z_n\times \Z_m\) is cyclic if and only if \(n,m\) are relatively prime. Here's the proof. Let \(L\) denote the least common multiple of \(n,m\text{,}\) so that \(L\leq nm\) and \(L=nm\) if and only if \(n,m\) are relatively prime. Next, observe that an element \((a,b)\) of \(\Z_n\times \Z_m\) is a generator if and only if every entry in the sequence
\begin{equation*} (a,b),(2a,2b),(3a,3b),\ldots,(nma,nmb) \end{equation*}
is distinct. If \(L\lt mn\text{,}\) then \((La,Lb)=(0,0)=(nma,nmb)\text{,}\) so \((a,b)\) is not a generator. This proves the "only if" direction. Conversely, suppose \(L=nm\) (that is, \(n,m\) are relatively prime). We claim that \((1,1)\) is a generator. If not, then there exist integers \(k,\ell\) in the range \(0\lt k,\ell\lt nm\) such that \(k\neq \ell\) and \((k,k)=(\ell,\ell)\text{.}\) Suppose, without loss of generality, that \(k\lt \ell\text{.}\) Then we have \((0,0)=(\ell-k,\ell-k)\text{,}\) so \(\ell-k\) is a integer number that is less than \(nm\text{,}\) and is divisible by both \(n\) and \(m\text{.}\) But this contradicts the assumption that the least common multiple of \(n,m\) is \(nm\text{.}\) This concludes the proof of the "if" direction, and the proof is complete.
