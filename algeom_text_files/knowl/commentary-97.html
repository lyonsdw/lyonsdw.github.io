Instructor's solution for Checkpoint 3.2.5.

It is straightforward to check that \({\mathcal T}\) is a homomorphism; half the work is already done in the solution to Checkpoint 3.2.3. It is easy to see that the scalar matrices are contained in \(\ker{\mathcal T}\text{.}\) To see that \(\ker{\mathcal T}\) is contained in the scalar matrices, observe the Möbius transformation \(\left[z\to \frac{az+b}{cz+d}\right]\) is the identity if and only if \(a=d\neq 0\) and \(b=c=0\text{.}\)
