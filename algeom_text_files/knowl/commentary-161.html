Instructor's solution for Exercise 3.6.1.

No, the two geometries are not equivalent. The transformation \((x,y)\to (x,-y)\text{,}\) corresponding to the orthogonal matrix \(\twotwo{1}{0}{0}{-1}\text{,}\) is an element of the group given in Subsection 3.1.1, but not in \(\E\text{.}\) To justify the "not in \(\E\)" claim, observe that generating elements of \(\E\) (rotations and translations) preserve oriented angles, so all elements of \(\E\) preserve oriented angles. But \((x,y)\to (x,-y)\) takes the oriented angle \(\angle 1,0,i\text{,}\) which measures \(+\pi/2\) radians, to the angle \(\angle 1,0,-i\text{,}\) which measures \(-\pi/2\) radians.
