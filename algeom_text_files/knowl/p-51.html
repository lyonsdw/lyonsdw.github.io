A mathematical model for one type of interaction dynamic is a function of the form \(f\colon S\times T \to U\text{.}\) A pair \((s,t)\) of objects interact to "create" an object \(f(s,t)\text{.}\) When \(S=T=U\text{,}\) we call a function \(f\colon S\times S \to S\) a binary operation on the set \(S\text{,}\) and we might view such a function as an interaction among objects in set \(S\text{.}\) A group is a set \(G\text{,}\) together with a binary operation \(G\times G \to G\text{.}\) Before we give the details of the required properties of a group's binary operation (see SectionÂ 1.2), the remainder of this section introduces examples of some important and useful groups.
