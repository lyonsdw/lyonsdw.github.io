Proof

Let \(f\colon X\to Y\) be a function. It is clear that the associated relation \(\sim_f\) is an equivalence relation. (Again, proving the three properties would be painful because of the the obviousness. You would have to say things like "if \(f(x)=f(y)\) then \(f(y)=f(x)\text{,}\) so \(\sim_f\) is symmetric".) The equivalence classes of \(\sim_f\) are clearly the preimages \(f^{-1}(\{y\})\) of 1-point sets \(\{y\}\subseteq Y\text{.}\) The function \(X/\!\!\sim_f\to f(X)\) given by \([x]\to f(x)\) is clearly onto. (You can justify the existence of \(f\) using FactÂ 1.4.2.) To see that this function is also one-to-one, just note that if \(f(x)=f(y)\) then \([x]=[y]\text{,}\) by definition of \(\sim_f\text{.}\)
