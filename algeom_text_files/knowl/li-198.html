Item b.

For reflexivity, we have \(C_{e}(x) = exe^{-1}=x\text{,}\) so \(x\sim x\) for all \(x\text{.}\) For symmetry, if \(y=C_a(x)=axa^{-1}\text{,}\) then \(C_{a^{-1}}(y)=a^{-1}ya=x\text{,}\) so \(x\sim y\) implies \(y\sim x\text{.}\) For transitivity, if \(C_a(x)=y\) and \(C_b(y)=z\text{,}\) then \(C_{ba}(x)=baxa^{-1}b^{-1}=byb^{-1}=z\text{,}\) so if \(x\sim y\) and \(y\sim z\text{,}\) then \(x\sim z\text{.}\)
