Using the distributive law, we have

\begin{equation*} R_r(v+w)=r(v+w)r^\ast = rvr^\ast +rwr^\ast = R_r(v)+R_r(w). \end{equation*}

Using the fact that real numbers commute multiplicatively with all quaternions, we have

\begin{equation*} R_r(\alpha v)= r(\alpha v)r^\ast=\alpha rvr^\ast=\alpha R_r(v). \end{equation*}
