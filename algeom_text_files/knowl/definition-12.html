Definition 2.3.5. Subgroups generated by a set of elements.

Let \(S\) be a subset of a group \(G\text{.}\) We write \(\langle S\rangle\) to denote the smallest subgroup of \(G\) that contains \(S\text{,}\) where smallest means that if \(H\) is a subgroup of \(G\) that contains \(S\text{,}\) then \(H\) contains \(\langle S\rangle\text{.}\) We call \(\langle S\rangle\) the subgroup of \(G\) generated by \(S\text{,}\) and we call the elements of \(S\) the generators of \(\langle S\rangle\text{.}\) [Comment on notational convention: If \(S=\{s_1,s_2,\ldots,s_k\}\) is finite, we write \(\langle s_1,s_2,\ldots,s_k\rangle\) for \(\langle S\rangle\text{,}\) instead of the more cumbersome \(\langle \{s_1,s_2,\ldots,s_k\}\rangle\text{.}\)]
