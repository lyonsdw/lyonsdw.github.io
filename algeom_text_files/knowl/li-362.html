Item b.

On \(S^2\text{,}\) points \(p,q\) are antipodal if and only if \(q=-p\text{.}\) So we have \(X'=-X\) for \(X=A,B,C\text{.}\) The antipodal map \(x\to -x\) is not an element of \(\Rot(S^2)\text{,}\) but it is clear that it is distance and area preserving. Triangles \(\triangle ABC\) and \(\triangle A'B'C'\) are antipodal images of each other, so they have the same area.
