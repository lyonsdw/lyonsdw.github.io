We have

\begin{align*} s(a,b,c)(s(-a,-b,-c))^\ast \amp = \frac{a+bi}{1-c}\left(\frac{-a-bi}{1+c}\right)^\ast\\ \amp = \frac{(a+bi)(-a+bi)}{1-c^2}\\ \amp = \frac{-a^2-b^2}{1-c^2}\\ \amp = \frac{c^2-1}{1-c^2} = -1. \end{align*}

Conversely, if \(zw^\ast = -1\text{,}\) then

\begin{align*} s^{-1}(w)\amp = s^{-1}\left(\frac{-1}{z^\ast}\right)\\ \amp = s^{-1}\left(\frac{-z}{|z|^2}\right)\\ \amp = \left(\frac{1}{|z|^2+1}\right) \left(\frac{-2\re(z)}{|z|^2},\frac{-2\im(z)}{|z|^2},\frac{1}{|z|^2}-1 \right)\\ \amp = \left(\frac{1}{1+|z|^2}\right) \left(-2\re(z),-2\im(z),1-|z|^2\right)\\ \amp = -s^{-1}(z). \end{align*}
