Instructor’s solution for Checkpoint 3.4.6.

The results of Exercise 1.2.6.2 imply that \(\Rot(S^2)\) is a group with identity element \(R_1\) and inverse given by \((R_r)^{-1}=R_{r^\ast}\text{.}\) Now let \(r=\cos \theta + u\sin\theta \) for some pure quaternion \(u\text{.}\) Proposition 1.2.9 says that \(R_r\) is a rotation by \(2\theta\text{,}\) so if \(R_r=R_1\text{,}\) then we must have \(\theta = k\pi\) for some \(k\in \Z\text{.}\) Therefore \(r=\cos(k\pi) = \pm 1\text{,}\) as claimed.
