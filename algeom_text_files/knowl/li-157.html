Item c.
Let \(p=|G|\text{.}\) By part 1, the order of any element in \(G\) must divide \(p\text{,}\) and since \(p\) is prime, the order of any element in \(G\) must be 1 or \(p\text{.}\) The identity is the only element of order 1, so the rest of the elements in \(G\) are all generators, and \(G\) is cyclic.
