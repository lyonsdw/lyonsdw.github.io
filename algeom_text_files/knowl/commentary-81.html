Instructor's solution for Exercise 2.6.1.
  1. Integers \(x,n\) are relatively prime if and only if \(\gcd(x,n)=1\text{,}\) if and only if there exist integers \(s,t\) such that \(sx+tn=1\) (see Exercise 2.3.2.4), if and only if \(sx=1 \pmod{n}\text{.}\)
  2. Multiplication is a well-defined binary operation on \(\Z_n\) (see Exercise 1.4.5.3) that is clearly commutative and associative. The element \(1\in U_n\) is the identity element, and the existence of inverse elements is part of the definition of \(U_n\text{.}\)
  3. Let \(f\colon \Z_n\to \Z_n\) be a group homomorphism, and let \(a=f(1)\text{.}\) Then
    \begin{equation*} f(x) = f(\underbrace{1+1+\cdots +1}_{x \text{ summands}})=xf(1)=ax \end{equation*}
    for all \(x\text{.}\) The homomorphism \(\mu_a\colon \Z_n\to\Z_n\) given by \(\mu_a(x)=ax\) is invertible if and only if \(a\in U_n\) (if \(a\not\in U_n\text{,}\) then \(1\) is not in the image of \(\mu_a\text{,}\) and conversely if \(a\in U_n\text{,}\) then \(\mu_a\) is invertible with inverse \((\mu_a)^{-1}=\mu_{a^{1}}\)). Thus we have a one-to-one correspondence \(U_n\leftrightarrow \Aut(\Z_n)\) given by \(a\leftrightarrow \mu_a\text{.}\) The fact that \((ab)x = a(bx)\) verifies that \(\mu_a\circ \mu_b=\mu_{ab}\text{,}\) so the correspondence \(a\leftrightarrow \mu_a\) is an isomorphism of groups.
