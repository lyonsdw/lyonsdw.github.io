Now we prove the Claim. Let \(b=\langle \lcm(a,n)/a\rangle\text{.}\) We have
\begin{equation*} \mu_a(b)=ab = \lcm(a,n) = 0 \pmod{n} \end{equation*}
so \(b\in \ker(\mu_a)\text{.}\) If \(c\in \ker(\mu_a)\text{,}\) then \(\mu_a(c)=ac=kn\) for some integer \(k\text{.}\) Since \(kn\) is divisible by \(a\text{,}\) we have \(kn\geq \lcm(a,n)\text{,}\) so \(c\geq b\text{.}\) This shows \(b\) is the smallest positive element in \(\ker(\mu_a)\) (or \(b=0\)).
