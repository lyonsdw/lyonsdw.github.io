Solve the system of equations
\begin{align*} r\amp =\frac{x}{1-y}\\ 1\amp =x^2+y^2 \end{align*}
for \(x\) and \(y\) (substitute \(x=r(1-y)\) into the second equation, then solve the quadratic for \(y\)) to obtain
\begin{align*} x \amp =\frac{2r}{r^2+1}\\ y \amp =\frac{r^2-1}{r^2+1} \end{align*}
so that we have
\begin{equation*} s^{-1}(r) = \begin{cases} \left(\frac{2r}{r^2+1},\frac{r^2-1}{r^2+1}\right) \amp \text{ if } r\neq \infty\\ (0,1)\amp \text{ if } r=\infty \end{cases}. \end{equation*}
Evaluating, we find \(s^{-1}(3) = (3/5,4/5)\text{.}\)
