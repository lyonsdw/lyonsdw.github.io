A function is surjective, or onto, if every element of the codomain has at least one preimage, that is, if every element in the codomain is the right entry of at least one element of the function. A function is injective, or one-to-one, if every element of the codomain has at most one preimage, that is, if every element in the codomain is the right entry no more than one element of the function. A function is bijective if it is both injective and surjective, that is, if every element in the codomain is the right entry of exactly one element of the function. A bijective function is also called a one-to-one correspondence.
