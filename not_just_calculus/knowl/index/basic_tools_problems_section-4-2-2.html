Exercise 2.
Let \(S\) be a set. An algebra of subsets of \(S\) is a subset \({\mathcal A}\subseteq {\mathcal P}(S)\) of the power set of \(S\) (see ExerciseÂ 1.5.3.1 above) for which the following properties hold.
  1. \(\displaystyle \emptyset \in {\mathcal A}\)
  2. For every \(X,Y\in {\mathcal A}\text{,}\) \(X\cap Y\in {\mathcal A}\)
  3. For every \(X,Y\in {\mathcal A}\text{,}\) \(X\cup Y\in {\mathcal A}\)
  4. For every \(X\in {\mathcal A}\text{,}\) \(S\setminus X \in {\mathcal A}\)
  1. Show that the power set \({\mathcal P}(S)\) is an algebra of sets for any set \(S\text{.}\)
  2. Given an example of a collection of sets of some set \(S\) for which exactly three of the four set algebra properties hold.
