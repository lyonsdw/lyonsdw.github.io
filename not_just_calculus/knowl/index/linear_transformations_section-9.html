If the matrix for \(L\) is \(\twotwo{a}{b}{c}{d}\text{,}\) and the matrix for \(M\) is \(\twotwo{e}{f}{g}{h}\text{,}\) then the matrices (the plural of matrix is matrices) for the linear maps \(\alpha L\text{,}\) \(L+M\text{,}\) and \(LM\) are given by
\begin{align} (\text{matrix for } \alpha L) \amp = \twotwo{\alpha a}{\alpha b}{\alpha c}{\alpha d}\tag{2.2.10}\\ (\text{matrix for } L+M) \amp = \twotwo{a+e}{b+f}{c+g}{d+h}\tag{2.2.11}\\ (\text{matrix for } LM) \amp = \twotwo{ae+bg}{af+bh}{ce+dg}{cf+dh}\tag{2.2.12} \end{align}
These natural constructions motivate the following matrix operations, called matrix scalar multiplication, matrix addition, and matrix multiplication, respectively.
\begin{align} \alpha \twotwo{a}{b}{c}{d} \amp = \twotwo{\alpha a}{\alpha b}{\alpha c}{\alpha d}\tag{2.2.13}\\ \twotwo{a}{b}{c}{d} + \twotwo{e}{f}{g}{h} \amp = \twotwo{a+e}{b+f}{c+g}{d+h}\tag{2.2.14}\\ \twotwo{a}{b}{c}{d}\twotwo{e}{f}{g}{h} \amp = \twotwo{ae+bg}{af+bh}{ce+dg}{cf+dh}\tag{2.2.15} \end{align}
Here is a way to remember the matrix multiplication operation. Entries in a matrix are labeled by rows and columns, starting from the upper left and counting from 1 to 2. For example, \(c\) is the row 2, column 1 entry in the matrix for \(L\text{,}\) above. In the product of the matrices for \(L\) and \(M\text{,}\) the entry in row \(i\) and column \(j\) is the dot product of row \(i\) in the matrix for \(L\) with column \(j\) in the matrix for \(M\text{.}\) For example, the row 1, column 2 entry in the matrix for \(LM\) is the dot product of row 1 in the matrix for \(L\) and column 2 in the matrix for \(M\text{.}\)
\begin{equation*} (a,b)\cdot (f,h) = af+bh \end{equation*}
