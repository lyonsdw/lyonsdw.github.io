A function is injective, or one-to-one, if no element of the codomain has more than one preimage, that is, if no element in the codomain is the right entry of more than one element of the graph.
 1 
If \((s,t)\) and \((s',t)\) are elements in the graph of an injective function, then it must be the case that \(s=s'\text{.}\) Conversely if \(s=s'\) for every pair of elements of the form \((s,t),(s',t)\) in the graph of a function, then that function is injective.
A function is surjective, or onto, if every element of the codomain has at least one preimage, that is, if every element in the codomain is the right entry of at least one element of the graph. A function is bijective if it is both injective and surjective, that is, if every element in the codomain is the right entry of exactly one element of the graph. A bijective function is also called a one-to-one correspondence.
