Exercise 1.

A graph is a pair of sets \((V,E)\text{,}\) where \(V\) is a set and \(E\) is a set of 2-element subsets of \(V\text{.}\) An element of \(V\) is called a vertex (plural: vertices) and an element of \(E\) is called an edge. A path from vertex \(v\) to vertex \(v'\) is a finite list of edges \(e_1,e_2,\ldots,e_k\) so that \(v\in e_1\text{,}\) \(v'\in e_k\text{,}\) and \(e_i\cap e_{i+1}\neq \emptyset\) for every \(i\) in the range \(1\leq i\lt k\text{.}\) A set \(W\subseteq V\) of vertices is connected if there is a path from \(w\) to \(w'\) for every pair of vertices \(w,w'\in W\text{.}\) A set \(W\subseteq V\) of vertices is a connected component of the graph if \(W\) is connected and \(W\) is not a subset of any other connected set of vertices. Show that the connected components of a graph form a partition of the vertex set \(V\text{.}\)
