    Section 2.2 Linear Mappings

    A function \(L\colon \R^2\to \R^2\) is called a linear mapping (or linear map) if \(L\) respects the operations of vector addition and scalar multiplication, that is, if
    \begin{align} L({\mathbf u}+{\mathbf v}) \amp = L({\mathbf u})+L({\mathbf v}), \text{ and }\tag{2.2.1}\\ L(\alpha{\mathbf u}) \amp = \alpha L({\mathbf u})\tag{2.2.2} \end{align}
    for all \({\mathbf u},{\mathbf v}\in \R^2\) and all \(\alpha\in \R\text{.}\) It is customary to omit the parentheses in the expression \(L({\mathbf u})\) and simply write \(L{\mathbf u}\) to denote the value of a linear mapping. With this convention, the linearity properties (2.2.1) and (2.2.2) look like this:
    \begin{align*} L({\mathbf u}+{\mathbf v}) \amp = L{\mathbf u}+L{\mathbf v}, \text{ and }\\ L(\alpha{\mathbf u}) \amp = \alpha L{\mathbf u} \end{align*}

    Checkpoint 2.2.1.

    For each of the following functions, determine whether or not it is linear. If a given function is linear, show that it satisfies properties (2.2.1) and (2.2.2). If a given function is not linear, give a specific example to show that it fails at least one of the properties (2.2.1) or (2.2.2).
    1. the function that sends every input point to its reflection across the \(x\)-axis
    2. the function \(g\) given by \(g(a,b)=(a+b,\sqrt{a^2+b^2})\)
    3. the constant function that sends every input point to the origin
    4. the constant function that sends every input point to the point \((1,0)\)
    5. the function \(h\) given by \(h(a,b)=(a+b,a-b)\)
    6. the function that sends input vector \(\mathbf{x}\) to output \(\mathbf{x}+\mathbf{b}\text{,}\) where \(\mathbf{b}\) is a constant nonzero vector
    If \(L\) is linear, and \({\mathbf u}=(x,y)\) is an arbitrary input value, then
    \begin{align} L{\mathbf u} \amp = L(x{\mathbf e}_1 + y{\mathbf e}_2)\tag{2.2.3}\\ \amp = L(x{\mathbf e}_1) + L(y{\mathbf e}_2)\tag{2.2.4}\\ \amp = xL{\mathbf e}_1 + yL{\mathbf e}_2.\tag{2.2.5} \end{align}
    If we write \(L{\mathbf e_1} = (a_1,b_1)\) and \(L{\mathbf e_2} = (a_2,b_2)\text{,}\) then the last expression (2.2.5) above becomes
    \begin{equation*} L{\mathbf u}=x(a_1,b_1)+y(a_2,b_2)=(a_1x+a_2y,b_1x+b_2y). \end{equation*}
    Thus, a linear mapping has a particularly simple form, and is summarized by four real constants \(a_1,b_1,a_2,b_2\) that are the components of the two values \(L{\mathbf e}_1,L{\mathbf e}_2\text{.}\) The square array of numbers \(\twotwo{a_1}{a_2}{b_1}{b_2}\) is called the matrix for \(L\text{,}\) and the array \(\begin{bmatrix} x\\y\end{bmatrix}\) is called the column vector that represents \({\mathbf u}=(x,y)\text{.}\) We write
    \begin{equation} L{\mathbf u} = \twotwo{a_1}{a_2}{b_1}{b_2}\begin{bmatrix} x\\y\end{bmatrix} =(a_1x+a_2y,b_1x+b_2y).\tag{2.2.6} \end{equation}

    Checkpoint 2.2.2.

    1. Suppose that the matrix for a linear map \(L\) is \(\twotwo{2}{1}{1}{-1}\text{.}\) Find \(L(3,2)\text{.}\)
    2. Suppose that a linear map \(L\) satisfies \(L(2,0)=(3,4)\) and \(L(0,3)=(2,1)\text{.}\) Find the matrix for \(L\text{.}\)

    Checkpoint 2.2.3.

    Find the matrix for each of the linear functions in Checkpoint 2.2.1.
    Given linear maps \(L,M\colon \R^2\to \R^2\) and a real number \(\alpha\text{,}\) we define linear maps \(\alpha L\text{,}\) \(L+M\) and \(LM\) by
    \begin{align} (\alpha L){\mathbf u} \amp = \alpha (L{\mathbf u})\tag{2.2.7}\\ (L+M){\mathbf u} \amp = L{\mathbf u} + M{\mathbf u}\tag{2.2.8}\\ (LM){\mathbf u} \amp = (L\circ M){\mathbf u} = L(M{\mathbf u}).\tag{2.2.9} \end{align}

    Checkpoint 2.2.4.

    Verify that \(\alpha L\text{,}\) \(L+M\text{,}\) and \(LM\) are indeed linear maps, that is, that they satisfy properties (2.2.1) and (2.2.2).
    If the matrix for \(L\) is \(\twotwo{a}{b}{c}{d}\text{,}\) and the matrix for \(M\) is \(\twotwo{e}{f}{g}{h}\text{,}\) then the matrices (the plural of matrix is matrices) for the linear maps \(\alpha L\text{,}\) \(L+M\text{,}\) and \(LM\) are given by
    \begin{align} (\text{matrix for } \alpha L) \amp = \twotwo{\alpha a}{\alpha b}{\alpha c}{\alpha d}\tag{2.2.10}\\ (\text{matrix for } L+M) \amp = \twotwo{a+e}{b+f}{c+g}{d+h}\tag{2.2.11}\\ (\text{matrix for } LM) \amp = \twotwo{ae+bg}{af+bh}{ce+dg}{cf+dh}\tag{2.2.12} \end{align}
    These natural constructions motivate the following matrix operations, called matrix scalar multiplication, matrix addition, and matrix multiplication, respectively.
    \begin{align} \alpha \twotwo{a}{b}{c}{d} \amp = \twotwo{\alpha a}{\alpha b}{\alpha c}{\alpha d}\tag{2.2.13}\\ \twotwo{a}{b}{c}{d} + \twotwo{e}{f}{g}{h} \amp = \twotwo{a+e}{b+f}{c+g}{d+h}\tag{2.2.14}\\ \twotwo{a}{b}{c}{d}\twotwo{e}{f}{g}{h} \amp = \twotwo{ae+bg}{af+bh}{ce+dg}{cf+dh}\tag{2.2.15} \end{align}
    Here is a way to remember the matrix multiplication operation. Entries in a matrix are labeled by rows and columns, starting from the upper left and counting from 1 to 2. For example, \(c\) is the row 2, column 1 entry in the matrix for \(L\text{,}\) above. In the product of the matrices for \(L\) and \(M\text{,}\) the entry in row \(i\) and column \(j\) is the dot product of row \(i\) in the matrix for \(L\) with column \(j\) in the matrix for \(M\text{.}\) For example, the row 1, column 2 entry in the matrix for \(LM\) is the dot product of row 1 in the matrix for \(L\) and column 2 in the matrix for \(M\text{.}\)
    \begin{equation*} (a,b)\cdot (f,h) = af+bh \end{equation*}
    The identity function \(L{\mathbf u} = {\mathbf u}\) is linear, and the matrix \(\twotwo{1}{0}{0}{1}\) for the identity function is called the identity matrix. It is easy to check that
    \begin{equation} \twotwo{a}{b}{c}{d}\twotwo{d}{-b}{-c}{a} = \twotwo{d}{-b}{-c}{a}\twotwo{a}{b}{c}{d} = \twotwo{ad-bc}{0}{0}{ad-bc}.\tag{2.2.16} \end{equation}
    It follows that, if \(ad-bc\neq 0\text{,}\) that a linear mapping \(L\) with matrix \(\twotwo{a}{b}{c}{d}\) is invertible, and that the matrix for \(L^{-1}\) is \(\frac{1}{ad-bc}\twotwo{d}{-b}{-c}{a}\text{.}\) This motivates the definition of the inverse of a matrix, denoted using the exponent \(-1\text{.}\)
    \begin{align} \twotwo{a}{b}{c}{d}^{-1} = \frac{1}{ad-bc}\twotwo{d}{-b}{-c}{a} \amp \amp (\text{ if } ad-bc\neq 0)\tag{2.2.17} \end{align}
    Comment: The quantity \(ad-bc\) is called the determinant of the matrix \(\twotwo{a}{b}{c}{d}\text{,}\) denoted \(\det\left(\twotwo{a}{b}{c}{d}\right)\).

    Checkpoint 2.2.5.

    1. Verify that the identity function \(\Id\colon \R^2\to \R^2\) is linear, and that the matrix for the identity function is \(\twotwo{1}{0}{0}{1}\text{.}\)
    2. Verify each of the equalities in (2.2.16).
    3. Explain why "it follows that \(\ldots\)" in the sentence immediately following (2.2.16).

    1.

    Verify the matrix formulas (2.2.10), (2.2.11), and (2.2.12)
    Hint.
    To obtain the matrix for a linear map \(L\text{,}\) use the fact that the first column of the matrix is the column vector \(L\mathbf{e}_1\text{,}\) and the second column is the column vector \(L\mathbf{e}_2\text{.}\)

    2. Distributive law for linear functions.

    Prove the following property of linear mappings, called the distributive law. For linear mappings \(K,L,M\) from \(\R^2\) to \(\R^2\text{,}\) we have
    \begin{equation} K(L+M)=KL+KM.\tag{2.2.18} \end{equation}

    3. Solving linear systems of equations.

    A pair of equations of the form
    \begin{align} ax + by \amp = r\tag{2.2.23}\\ cx + dy \amp = s\notag \end{align}
    where \(a,b,c,d,r,s\) are constants, and \(x,y\) are unknowns, is called a linear system of equations in two unknowns. A solution is a pair of values for \(x,y\) so that both equations in (2.2.23) hold. We can use matrix algebra to solve linear systems of equations, as follows. Using the notation defined in (2.2.6), here is the matrix version of (2.2.23).
    \begin{equation} \twotwo{a}{b}{c}{d}\begin{bmatrix}x\\y\end{bmatrix} =\begin{bmatrix}r\\s\end{bmatrix}\tag{2.2.24} \end{equation}
    The matrix \(A=\twotwo{a}{b}{c}{d}\) is called the matrix of coefficients for the linear system. If we write \(\mathbf{u}=\begin{bmatrix}x\\y\end{bmatrix}\) and \(\mathbf{v}=\begin{bmatrix}r\\s\end{bmatrix}\) then we can write (2.2.24) even more compactly as
    \begin{equation} A\mathbf{u} = \mathbf{v}\tag{2.2.25} \end{equation}
    If \(A\) is invertible, we can multiply both sides of (2.2.25) by \(A^{-1}\) on the left, and we have
    \begin{equation} \mathbf{u} = A^{-1}A\mathbf{u} = A^{-1}\mathbf{v}.\tag{2.2.26} \end{equation}
    Use this matrix method to solve the following linear system. Find \(A\text{,}\) \(A^{-1}\text{,}\) \(x\text{,}\) and \(y\text{.}\)
    \begin{align*} 3x - y \amp = -2\\ x + 2y \amp = 3 \end{align*}

    4. Rotations.

    Let \(R_\theta\colon \R^2 \to \R^2\) be the counterclockwise rotation by \(\theta\) radians about the origin, that is, \(R_\theta(r\cos \phi,r\sin\phi)=(r\cos(\phi+\theta),r\sin(\phi+\theta))\text{.}\)
    1. Show that \(\R_\theta\) is linear for all \(\theta\text{.}\)
    2. Show that the matrix for counterclockwise rotation by \(t\) radians about the origin is \(\twotwo{\cos t}{-\sin t}{\sin t}{\cos t}\text{.}\)
    Hint.
    It will be useful to use the angle sum formulas:
    \begin{align*} \cos (\theta + \phi) \amp = \cos \theta \cos \phi - \sin\theta \sin \phi\\ \sin(\theta+\phi) \amp = \cos \theta \sin\phi + \sin\theta\cos\phi \end{align*}

    5. Reflections.

    1. Show that the reflection \(F_X\) across the \(x\)-axis is linear.
    2. Find the matrix for \(F_X\text{.}\)
    3. Let \(\ell\) be a line through the origin. Show that reflection \(F_\ell\) across the line \(\ell\) can be written as a composition \(R_{\theta}\of F_X \of R_{-\theta}\) for some appropriate value of \(\theta\text{.}\) Find the matrix for \(F_\ell\text{.}\)
